## 📁 Projetos em Destaque

### ⚖️ Sistema Inteligente de Minutas Judiciais com IA
Uma aplicação corporativa ponta a ponta projetada para otimizar rotinas jurídicas complexas. O sistema realiza o processamento em lote de múltiplos arquivos PDF de forma sequencial, utilizando visão computacional e inteligência artificial multimodal (Gemini API) para transcrever e correlacionar autos judiciais escaneados sem o uso de dependências locais de OCR.
* **Stack:** Python, Streamlit, Supabase (PostgreSQL), PyMuPDF, Bcrypt, python-docx.
* **Destaques de Engenharia:** Proteção rigorosa contra vazamento de dados por meio de rotinas automatizadas de eliminação de resíduos temporários (Conformidade LGPD), barramento de estado resiliente e criptografia padrão ouro com Salt dinâmico para gerenciamento de usuários.

### 🛍️ E-Commerce & Plataforma de Vestuário
Projeto prático e funcional focado no desenvolvimento de uma aplicação web completa para a gestão de uma loja de vestuário. O sistema integra interface do usuário intuitiva a um back-end robusto focado em controle de inventário, fluxo de pedidos e persistência estruturada de dados.
* **Stack:** C#, .NET, SQL Server / Python.
